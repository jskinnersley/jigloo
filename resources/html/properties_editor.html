The GUI Properties Editor

The GUI Properties editor enables you to change the properties of single or multiply-selected form components. It comprises three separate editors:
  1. One for editing the properties of the selected GUI component(s)
  2. One for changing the layout and layout constraints for the selected component(s)
  3. One for setting event listeners for the selected component(s)
If multiple components are selected then only those properties which the components have in common will be shown, and setting a property/event/layout will set it for all selected components.

For boolean properties, the default behaviour is to toggle between true and false if the checkbox is clicked. This single-click behaviour can be changed to a double-click by a setting in the Preferences Page.
